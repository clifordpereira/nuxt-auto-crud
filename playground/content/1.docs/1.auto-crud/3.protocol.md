# NAC API Protocol

The Core Module (`nac`) supports server-side orchestration. Agents should prioritize specific queries over bulk fetching.

### Current Implementation Status
- **Search:** Client-side via `Object.values(row).join(' ')`. (Note: Inefficient for large datasets).
- **Pagination:** Client-side slice.
- **Future Protocol:** Move to `useFetch(url, { query: { q, page, limit } })`.
